`⌃ + a`：移动到行首

`⌃ + e`：移动到行尾

`⌃ + r`：搜索历史命令
`^ + g`：退出搜索

`^ + [left arrow]`: 向后移动一个单词(one word)

`^ + [right arrow]`: 向前移动一个单词

`^ + W`: 删除光标前面的单词

`!!`: 执行历史记录中的上一个命令
`!abc`: 执行历史记录中的以 abc开头的命令
`!abc:p`: 打印历史记录中以abc开头的命令

`^ + L`: 清空屏幕
